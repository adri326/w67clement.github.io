Events system

What is the events system?
The events system is a simple system like Bukkit's Events for detect the sent packets.
But how to use this?

Theoretical course

Hum, wow, we see how to use this. :D
First, we don't use the Bukkit's Listener interface! We use the PacketListener
And, other thing, we don't use the EventHandler annotation, but the PacketHandler!
Now, we need to know many things, the PacketHandler have argument: PacketList listenType. the listenType define what is the packet to listen.
The PacketHandler annotation can be used only on methods, the methods need have as arguments a MineAPI Event, and he have 4:

  • SendPacketEvent
  • ReceivePacketEvent
  • PacketPingSendEvent
  • PacketPingReceiveEvent The ping packets are for the Status Packet, SendPacketEvent and ReceivePacketEvent are for Play packets.
    We can define a type argument on these arguments, but they need to extends NmsPacket.
    Now you can use the MineAPI's Events System.
