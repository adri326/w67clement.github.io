Json Configuration

Json is an format can be used for configurations.
With this, you can load your configurations easily.
But, how to use?

JsonConfig

It extends from Config class, in JsonConfig, you can find a JsonElement instance. You can find others things: JsonParser and Gson instances.
To load an configuration, you just need to call the method void load, it initializes the JsonElement with the content in the specified file.
To save an configuration, you just need to call the method void save, it write the file in Json with the content of the JsonElement.
Now you can create configurations in Json!
