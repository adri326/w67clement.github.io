Create custom configuration

You want another configuration type? Like Java Properties for example?
It's possible!

First, you need to know how to use the API for read and save files. After, you can start to create your custom configuration!
Now, create a new class who extends MineAPI's Config class. Make it abstract.
In this class, let's rewrite some methods!
First create an variable who have for value the instance of an API's Object (Example: JsonElement for Gson, FileConfiguration for Bukkit).
After, rewrite the method public void load(). In the method, write super.load();, it's for the save resource.
After super.load();, write the method to parse/read/load the configuration to the variable.
Now, you need to write the method public void save(), in this, write the method to save the configuration.

Wow! Congratulations, you know how to create an custom configuration now!
