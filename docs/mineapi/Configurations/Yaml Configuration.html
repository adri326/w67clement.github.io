Yaml Configuration

Yaml is an format for configurations, it's used by Bukkit API.
With this, you can load your configurations easily.
But, how to use?

YamlConfig class

It extends from Config class.
In this class, you can get the FileConfiguration instance of Bukkit API.
In the constructor you can call the method init and in the method init call the method load.
YamlConfig have two methods to get and set a Location from/in the configuration.
The location is formatted in Json, example:

[
{
    "world": "world",
    "x": 0.0,
    "y": 64.0,
    "z": 0.0,
    "yaw": 0.0,
    "pitch": 0.0
}
]
And a example inline:

[{"world":"world","x":0.0,"y":64.0,"z":0.0,"yaw":0.0,"pitch":0.0}]
The methods getLocation and setLocation have the same methods of FileConfiguration for get and set data.
For get and set data, you just write in the method init, after load:

public void init()
{
    this.load();
    this.prefix = this.configuration.getString("mineplugin.prefix", "[MinePlugin]");
    this.spawnLocation = this.getLocation("mineplugin.spawn.location", new Location(Bukkit.getWorlds()[0], 0.0D, 64.0D, 0.0D));
}
Now, you know how to use the MineAPI's YamlConfig.
