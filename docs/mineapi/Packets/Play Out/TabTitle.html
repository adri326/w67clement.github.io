Using TabTitle

MineAPI offers the possibility to create tab titles and send them to the player, see how it works!

To create the new Title object you have to go through this code: 

MineAPI.getNmsManager().getTabTitle(Header, Footer)


Good! Detail it!

  • Header is an String value, sets header of the tab.
  • Footer is an String value, sets footer of the tab.

Note: You can use \n to create a new line.

Well, that's fine but how to send it to a player?

You will need to call the function send(Player player), this function will ask the player as argument. To send the title to all players at the same time, call the function sendAll().

Good! now see the other functions!

  • getHeader() returns the header texts.
  • setHeader(String header) sets the header texts.
  • getFooter() returns the footer texts.
  • setFooter(String footer) sets the footer texts.

Good! I give you a small example of use?

Yes?

The TabTitle header will be: "Welcome to my server" and it will be green and will footer be: "Good game!" and will also be green. The TabTitle will be visible to all players.

The code: 

MineAPI.getNmsManager().getTabTitle(ChatColor.GREEN + "Welcome to my server!", ChatColor.GREEN + "Good game!").sendAll();


That's it! You now know how to use the TabTitles without any difficulties! Thank you for reading!
