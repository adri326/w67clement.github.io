Jouez sur votre clé USB !


Cross-platform

Grâce au langage cross-platform Java, avec MineUSB sur votre clé USB, vous pourrez jouer en quelques clics sur n'importe quel système d'exploitation.
MineUSB détectera l'OS utilisé et choisira la meilleur solution.
Une seule chose sera nécessaire, une JVM !

Un launcher intégré !

Vous en avez marre du launcher par défaut ?
Vous voulez installer Forge ou Optifine sans quitter le launcher et en 2-3 clics ?
Et bien, le launcher intégré de MineUSB vous le permettra ! Une fois connecté, choisissez la version, utilisez Forge ou Optifine en quelques clics !

Configurable

MineUSB est très simple à configuré !
En 3 clics, changez la langue choisie!
En 3 clics, changez le launcher sélectionné !

Configuration et Dossiers

Tout d'abord voici une structure complète une fois MineUSB utilisé: 

| Dossier parent
|
|-- bin/
|  |-- game/ (Dossier généré sous Windows par le launcher MC)
|  |-- runtime/ (Dossier généré sous Windows par le launcher MC)
|  |- Minecraft.exe (Launcher Windows MC)
|  |- Minecraft.jar (Launcher Mac/Linux/Solaris MC)
|  |- (Autres launchers)
|
|-- data/
|  |-- .minecraft/ (Notre chèr dossier MC)
|  |-- .QuelqueChose/ (Dossier généré par les launchers)
|  |-- MineUSB/
|  |  |-- lang/ (Bientôt)
|  |  |-- logs/ (Dossier avec les logs)
|  |  |- config.json (Configuration)
|  |  |- MineUSB.jar (MineUSB)
|  |  |- Version.txt (Ne surtout pas modifier!)
|
|- MineUSB-Bootstrap-[$VERSION].jar

Bien, mais à quoi ressemble un config.json vierge ?

Vous pourrez le remarquer, il y a une section appelé auth, c'est pour le launcher intégré.
Le mot de passe est crypter, donc ne craigniez pas le vol de mot de passe, mais essayez de ne pas publier cette partie de la configuration si vous vous êtes déjà connecté avec votre compte Minecraft.

Téléchargements et sources

Vous pouvez télécharger le bootstrap qui mettra à jour automatiquement MineUSB. Les sources aussi peuvent être téléchargé puisque le projet est OpenSource.

Type Lien Version Conseillé Raison
Bootstrap MineUSB-Bootstrap-1.0.1.jar 1.0.1 Met à jour automatiquement.
MineUSB MineUSB_Beta_1.0.3.jar Beta 1.0.3 Pas d'update automatique.
Sources MineUSB - Github - For developpers only.
