MineAPI 1.6.0

MineAPI, késaquo?

MineAPI est un plugin pour les développeurs permettant d'utiliser les class Nms!

Il vous suffira d'utiliser le Nms Manager de MineAPI pour utiliser certaines class de Nms!

Depuis MineAPI 1.6.0, MineAPI offre la possibilité d'installer des modules! Mais c'est quoi? C'est un système pour installer des extensions de MineAPI!

Ainsi, il pourra offrir des possibilités que MineAPI ne pourrait pas offrir dans son .jar principal!

Le plugin dans sa version 1.5.0 ne marche que sous CraftBukkit/Spigot 1.8 - 1.8.7

Commandes et permissions

  • /AdvancedVersion Donne la version du serveur, du protocol et de Bukkit.

  • mineapi.* Donne toutes les permissions.
  • mineapi.cmd.* Donne toutes les permissions de toutes les commandes de MineAPI.
  • mineapi.cmd.advancedversion Permission de la commande /AdvancedVersion.
