AdvancedSignEdit 1.0.4

AdvancedSignEdit est un plugin Bukkit en Java, celui-ci va vous permettre d'éditer vos panneaux sur votre serveur CraftBukkit!

On peut l'éditer avec une commande ligne par ligne ou (pour la première fois) ouvrir l'interface d'édition de panneau!

Le plugin dans sa version 1.0.4 ne marche que sous CraftBukkit/Spigot 1.7.10 - 1.8

Il y a aussi différents symboles disponnibles:

[++] → ✦

<3 → ♥

Commandes et permissions

  • /EditSign open Ouvre l'interface du panneau visé.
  • /EditSign editligne [Ligne] [Texte] Change le text de la ligne choisi du panneau visé. À noter que les symboles marchent sur cette commande.

  • advancedsignedit.* Donne toutes les permissions.
  • advancedsignedit.bypass Ignore la liste des panneaux non-éditable.
  • advancedsignedit.cmd.editsign Permission de la commande /EditSign.
  • sign.* Donne les permissions sign.edit et sign.color
  • sign.edit Permission pour éditer un panneau. (Sans la commande)
  • sign.color Autorise les joueurs à utiliser de la couleur sur les panneaux avec le code couleur.

Configuration

Fichier config.yml:

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- 

#Configuration of AdvancedSignEdit

Config:
  #Lists of texts if present on line 1 of a not editable sign
  SignNotEditable:
    - '&1[Heal]'
    - '&4[Heal]'
    - '&1[Gamemode]'
    - '&4[Gamemode]'
    - '&1[Free]'
    - '&4[Free]'
    - '&1[Disposal]'
    - '&4[Disposal]'
    - '&1[Time]'
    - '&4[Time]'
    - '&1[Weather]'
    - '&4[Weather]'

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fichier messages.yml:

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- 

#Change the different messages of the plugin!
Prefix:
  Usage: '&4[Usage] '
  Error: '&4[Error] '
  Permissions: '&4[Permissions] '

#The message is sent when the plugin open a sign editor
OpenGui:
  Message: '&aOpen GUI...'
  Chat: true
  ActionBarMessage: false

#The message is sent when the sign was edited successful
ChangeSuccess:
  Message: '&aThe sign was changed successful!'
  Chat: true
  ActionBarMessage: false

#The message is sent when the configuration was reloaded successful
ReloadSuccess:
  Message: '&aThe AdvancedSignEdit's configuration was reloaded successful!'
  Chat: true
  ActionBarMessage: false

#The message is sent when the block is not a sign
IsNotASign:
  Message: '&cThe block is not a sign!'
  HoverMessage: '&cSorry =/'
  Chat: true
  ActionBarMessage: false

#The messages is sent when player try edit a not editable sign.
SignNotEditable:
  Message: '&cThis sign is not editable!'
  Chat: true
  ActionBarMessage: false

InvalidNumber:
  Message: '&cThe number specified is invalid!'
  Chat: true
  ActionBarMessage: false

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
